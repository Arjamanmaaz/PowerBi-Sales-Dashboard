Executive Sales Overview Dashboard (Power BI)

This Power BI project provides a clear and meaningful analysis of an organization’s sales performance. It brings together essential KPIs, visuals, and trends to help decision-makers quickly understand business performance and identify opportunities for improvement.

Key KPIs Included
- Total Sales generated  
- Total number of Orders  
- Average Order Value (AOV)  
- Total Customers served  

Visuals Used
- Year-wise Revenue Trend to track sales growth  
- Store-wise Sales Comparison to identify top-performing locations  
- Category-wise Revenue Distribution (Donut Chart) to show product contributions  
- KPI Cards for quick highlights of business health  

Features
- Interactive slicers for filtering and exploring data  
- Clean and professional dashboard layout  
- Business-friendly insights designed for executives  
- Data cleaning, transformation, and modeling using Power Query and DAX  

Tools & Technologies Used
- Power BI Desktop  
- Power Query Editor  
- DAX  
- Data Modeling (Star Schema)
